# Sistema de Inventarios – Aplicación de Escritorio

## Descripción
Aplicación de escritorio desarrollada en **C# con Windows Forms** para la gestión de inventarios y ventas. Permite registrar productos, controlar stock, gestionar ventas y calcular totales automáticamente. Incluye múltiples pantallas y validación de acceso al sistema.  

> Nota: La aplicación **no está conectada a una base de datos**; los datos se manejan en arreglos dentro del programa.

## Funcionalidades
- Gestión de productos e inventarios
- Registro de ventas y cálculo automático de totales
- Sistema de login y validación de usuarios
- Interfaz gráfica con varias pantallas

## Tecnologías
- C#  
- Windows Forms  

## Cómo usar
1. Clona o descarga el repositorio.  
2. Abre el archivo `sistemainventarios.sln` con **Visual Studio**.  
3. Compila y ejecuta la aplicación.  

## Observaciones
- Carpetas `bin` y `obj` no están incluidas, ya que son generadas al compilar.
- Los datos se manejan en memoria usando arreglos; no requiere base de datos externa.
